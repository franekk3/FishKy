Supported Android Versions:

Minimum API required by app is API 21 (Android 5.0), and works fine up to newest Android 16.

"Modern UI" support:

Even if app supports Android 5.0 and newer, not all functions will work on that old version. "Modern UI" support is confirmed on Android 14+ (likely Android 12+).
